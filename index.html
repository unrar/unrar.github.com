UnRar's UnBot

UnRar's UnBot is an IRC bot project mainly designed to interact with Wikipedia, using MediaWiki's API. It's widely used on the #wikipedia-es and #wikipedia-es-bots, but can be ported to other languages or projects. It's proudly hosted by ClumsyFairyQueen's server.

You're free to download or fork a copy of UnBot, and port it to another project, by Wikimedia or by you; or to just fork the bot and adapt it to whatever you use. Please note that I'll only support the "main" UnBot, so if you modify something and have a problem with, let's say, the %ip command, I won't support it.

To "clone" the actual UnBot, run the following command:

$ git clone

You can have your clone always up-to-date by using this command on the directory you cloned:

$ git pull

Branches

We actually have 2 branches:

  • master - It's the branch for Sybil (actual stable-frozen).
  • shiva - Branch for the Shiva (testing) development.

In the future, we'll add more branches for the old-stable version, the unstable version...

Release schedule

We use a Debian-like release schedule. There are always a stable version, an unstable version and a testing version. They have a codename, actually Sybil (frozen), Peggy and Shiva. The unstable version shouldn't be used and its name will be always Peggy.

There are also "modifiers" for the releases. There's a stable-frozen version, currently Sybil. Being frozen means it won't be more longer supported, thus there won't be more bug fixes and no more issues will be solved. This status is used transitionally meanwhile we change from stable to testing. During the frozen or transition stage, we focus on the testing version, and to convert it to stable. Once it reached the stable status, the old stable-frozen version changes its codename and has the old-stable status. The old-stable releases aren't supported, but still avaliable to download or fork.
